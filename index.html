Basically, for the past few weeks, I have learnt about HTML and CSS.

I undersatnd that Html stands for Hypertext Markup Language. It is used to create web pages and it is also the structure of the webpage.The basic building blocks are tags,attributes and elements.

Tags surround the content and each tag performs a specific function. Examples include: P tag, div tag, b tag, img tag, table tag e.t.c. Tags are enclosed in 'arrow heads'. Most tags have a start and close tag but some are self closing like the img tag. Attributes give more information about a specific tag e.g the img tag has ''src atrribute, which states the source of the image from the file or a link from the internet and the 'alt' attribute, which describes the image incase the image is broken for the user to have an idea what the image is. Element includes the start tag, content and the end tag.The Html skeletion starts with DOCTYPE Html that lets the browser know the type of html. It consists of the head and body parts embedded in the html start and end tag. The body part contains all the things that we see on the webpage and it where we put all the elements of html. Also, the inline CSS can be put in the body part of the skeleton. The head part contains the title that shows as the title of the document on the browser,the link to external css or internal css.

CSS stands for Cascading Style Sheet. It is used for designing the web page alonside with Html. CSS could be external(has a separate file in the same folder but linked to the html file using which is placed in the head tag of the html),internal(uses the style tag embedded in the head of the html file) or inline style, which is written within the body of the html file. There are different selectors used in CSS like the universal, element, Id, class and attributeselectors. These selectors makes it easy to design the html as they implement the design to different parts of the file as one likes. Although there are CSS properties that make one selector or design format overide the other but they all work together to give a good look to the webpage. The color, font size,font style, text decoration, margins and other properties can be styled.

Paddings, borders, margins are part of the CSS box model that surrounds the content. From inside the Box Model out, padding is the space between the content and the border. The border is the line that is the between the padding and the margin and could be styled in different ways with border color, width , style and radius. Border color specifies the color of the border, the width specifies the size of the border and the style specifies the design of the line e.g solid, dashed, dotted e.t.c. The border radius is the property that gives rounded edges to the border.The margin is the space outside the border. The margin and padding could be styled top, bottom, left and right. Alternatively, margin and padding could be styled with equal spacing on all sides with one value.
